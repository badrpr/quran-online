# 📖 Al-Quran Online - Premium Reading Experience

Une application web moderne, fluide et élégante pour lire et écouter le Saint Coran en ligne. Conçue avec une esthétique "Glassmorphism" et une attention particulière portée à l'expérience utilisateur.

![Quran Online Preview](https://raw.githubusercontent.com/badrpr/quran-online/main/preview.png) *(Note: Remplacez ceci par une vraie capture d'écran une fois déployé)*

## ✨ Fonctionnalités

- **🌙 Design Premium** : Interface sombre élégante avec des accents émeraude et or, utilisant le système de design Glassmorphism.
- **📖 Lecture Ininterrompue** : Accès aux 114 sourates avec le texte arabe Uthmani original.
- **🇫🇷 Traduction Intégrée** : Traduction française complète pour chaque verset (Hamidullah).
- **🔊 Audio Dynamique** : Écoutez la récitation de Mishary Rashid Alafasy.
- **🎯 Auto-Scroll & Highlight** : La page défile automatiquement pour suivre la récitation et souligne le verset en cours.
- **🔍 Recherche Intelligente** : Trouvez rapidement une sourate par son nom (en français/anglais/arabe) ou par son numéro.
- **📱 Entièrement Responsive** : Optimisé pour une lecture parfaite sur mobile, tablette et ordinateur.

## 🚀 Installation Locale

Aucune installation complexe n'est requise. Ce projet utilise uniquement du HTML, CSS et JavaScript pur (Vanilla).

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/badrpr/quran-online.git
   ```
2. Ouvrez le dossier :
   ```bash
   cd quran-online
   ```
3. Ouvrez le fichier `index.html` dans votre navigateur préféré.

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 (Vanilla), JavaScript (ES6).
- **API** : [Al Quran Cloud](https://alquran.cloud/api) - Source de données fiable et rapide.
- **Fonts** : Google Fonts (Amiri pour l'Arabe, Inter pour le texte).
